Whenever a person starts learning computer science than the first language which is recommended to him to learn is C
C is a procedural programming programming language, It was developed by  "Dennis ritchie"
in 1963. C is a Low level language and has simple set of pre defined keywords, which makes it easier for begginers to understand

Let`s take a glance at our First Hello World program in C programming language.

#include
int main()
{
printf("HELLO WORLD!");
}


OUTPUT:-

In the above Code :-

  1. #include  :- is a header file which must be included in every C program as it contain various definition & function necessary for a C program to work correctly.


There are various other libararies in C such as:-
math.h
stdlib.h
string.h
and various other

2. main() :- Every C program should must have a main function to run a C program, untill unless we don`t go for user-defined function or other user-defined data type, Everything in the program is written inside main function.

3. printf() function :- printf is used to print value of integer , string , char , float etc on the console Screen.
